# Hex Tile Designer – Prior Versions

## Downloading a Version

Each release of Hex Tile Designer is a single self-contained `index.html` file.

To download a specific version:

1. Find the release you want in the [Releases list](https://github.com/AndyPryke/Apps/releases) (look for tags beginning with `hextiles/`).
2. Open the release and browse the source files, or use the tag to view the repository at that point in time.
3. Open `HexTiles/index.html` and save the page (or copy the raw file content) to your device.
4. Open the saved `index.html` in any modern browser — no internet connection required after that.
