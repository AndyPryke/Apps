## [0.1] - 2026-06-06

### Added

- Load webR and run `cat("Hello, R!")` in the browser, displaying the output in a terminal-style console.
- Name input: enter your name and R greets you personally using `cat(paste0("Hello, ", name, "!"))`.
- Iris pairs plot coloured by species, with a personalised title, generated in R and displayed below the greeting.
